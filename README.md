# Instruções para executar  o sh.
No terminal linux, use o comando ./calculadora.sh para executar o script. Siga as instruções para cálculos básicos. Esse projeto foi criado para exercitar a criação de arquivo no ambiente Linux, a alteração de permissões e execução se script. No arquivo de texto, há os comandos utilizados.

# Explicação do código em Python
Esse código foi criado para execução de uma calculadora simples, com o propósito para exercitar o comando "input", conversão de valores "int" e "float", implementação das quatro operações matemáticas, e condicionais.
